## HR Recruitment Analysis Project

## Project Overview

This project demonstrates a simple HR recruitment tracking system using Microsoft Excel.
It includes candidate evaluation and recruitment funnel analysis to understand the hiring process.

## Tools Used

* Microsoft Excel
* GitHub


## Project Files

* Candidate_Evaluation_Sheet.xlsx
* Hiring_Funnel_Data.xlsx


## Candidate Evaluation Method

Candidates are evaluated based on:

* Skills (out of 10)
* Communication (out of 10)
* Experience (out of 10)

Formula used in Excel:

=SUM(D2:F2)


This calculates the total candidate score.


## Recruitment Funnel Stages

The project analyzes recruitment stages:

* Applications
* Shortlisted
* Interviewed
* Selected
* Joined

A bar chart is created to visualize the recruitment funnel.


## Key Learning

* Candidate tracking in recruitment
* Interview evaluation using scoring system
* Recruitment funnel analysis using Excel
